SITE DE JOGOS

O projeto foi concebido e elaborado pelos estudantes do terceiro período do curso de Análise e Desenvolvimento de Sistemas do IFPE Campus Palmares, visando criar um site com links para jogos online destinado aos idosos do SCFVI.

DOMINÓ

Dominó é um jogo de mesa que utiliza peças com formatos retangulares, dotadas normalmente de uma espessura que lhes dá a forma de paralelepípedo, em que uma das faces está marcada por pontos indicando valores numéricos.

DAMAS

Jogo de damas ou simplesmente damas é um grupo de jogos de tabuleiro de estratégia para dois jogadores que envolvem movimentos diagonais de peças de jogo uniformes e capturas obrigatórias saltando sobre o oponente peças.

PACIÊNCIA

Paciência, também conhecido como Solitário, é um jogo de cartas para um só jogador. É um jogo muito famoso sobretudo por causa de uma versão para o sistema operacional Microsoft Windows, que é considerada um dos jogos para computador mais populares de todos os tempos e o videojogo mais jogado no mundo inteiro..

JOGO DA MÉMORIA

O jogo da memória é um clássico jogo formado por peças que apresentam uma figura em um dos lados. Cada figura se repete em duas peças diferentes. Para começar o jogo, as peças são postas com as figuras voltadas para baixo, para que não possam ser vistas.
